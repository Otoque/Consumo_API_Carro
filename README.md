# 🚗 Carros Explorer

Aplicação web simples que consome uma API de carros e exibe informações detalhadas sobre veículos diretamente no navegador.

O projeto foi desenvolvido utilizando **HTML, CSS e JavaScript**, com o objetivo de praticar consumo de APIs REST e manipulação de dados no frontend.

---

## 🛠 Tecnologias

- HTML
- CSS
- JavaScript
- Fetch API

## ⚙️ Como executar

### você pode acessar via web entrando no link

    https://consumo-api-carro.vercel.app/

### Ou rodar o projeto direto no seu computador

1. clone o repositório
   
   git clone https://github.com/Otoque/Consumo_API_Carro.git

2. Abra a pasta onde foi clonado o repositório

   cd Consumo_API_Carro
   
3. Abra o arquivo `index.html` no navegador.

---

## 🔎 Funcionamento

Ao clicar no botão **Buscar Carro**, a aplicação faz uma requisição para a API:

GET /carros/aleatório


Os dados retornados são exibidos na interface, incluindo:

- Informações do veículo
- Dados do motor
- Potência
- Combustível
- Fabricante
- Imagem do carro

---

## 📱 Responsividade

O projeto possui layout adaptado para:

- Desktop
- Tablet
- Celular
- No modo mobile:
- A imagem aparece primeiro
- As informações aparecem abaixo
- A navegação é vertical

## 🔗 API utilizada

O projeto consome dados da **API de Carros**, desenvolvida separadamente para fornecer as informações dos veículos.
